Cara buat perubahan permanen di GitHub:
1. Klik Simpan & Terapkan dulu
2. Klik Salin Config untuk GitHub
3. Buka GitHub → index.html → ikon pensil ✏️
4. Cari teks: // === KONFIGURASI UTAMA ===
5. Hapus seluruh blok CONFIG lama, paste yang baru
6. Klik Commit changes → Netlify auto update ✅
