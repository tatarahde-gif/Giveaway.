Cara setup EmailJS:
1. Daftar gratis di emailjs.com
2. Add Email Service (Gmail, dll) → dapat Service ID
3. Buat Email Template → dapat Template ID
4. Di template, gunakan variabel: {{username}}, {{reason}}, {{time}}
5. Settings → API Keys → copy Public Key
6. Paste semua di field di atas & Save!
